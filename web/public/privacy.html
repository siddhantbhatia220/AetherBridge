AetherBridge

Privacy Policy

Last updated: April 27, 2026

1. Data Collection

AetherBridge collects minimal data required to facilitate API requests between your application and third-party providers. In development mode (Shadow Kernel), all data stays on your local machine.

2. Security

All data transmitted through the AetherBridge Proxy is encrypted using industry-standard TLS. We do not store your API secrets; they are passed directly to the providers or managed via environment variables.

3. Local Storage

The Shadow Kernel may store mock data in a local JSON file (shadow-db.json) for persistence during development.
